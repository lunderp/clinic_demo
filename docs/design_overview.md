

# 始末诊所
### 一句话介绍游戏
你将扮演一名心理医生对患者进行问诊与治疗。

## 设计
始末诊所是一个**SLG大类下的剧情驱动游戏**，日常玩法是通过患者治疗，影响玩家的压力值、心情值、金钱，来决定游戏的不同走向。通过**各具特色的患者**来吸引玩家期待故事的发展，插入悬疑事件与**不协和音**来塑造悬疑氛围。设计了陈默作为游戏的主角，莫矢作为主角2。心理话题整体需要严肃压抑的氛围，因此美术上选取较为严肃的画风，粗外线少内线、色彩低饱和、尽量降低细节丰富度，以**减少玩家的大脑处理噪声信息的压力**，同时令莫矢性格活泼开朗，令部分患者行为夸张搞笑，来调节游戏氛围，**过于严肃就不是游戏，而是游戏作者的自我高潮**。

### 一阶段游戏一个医生模拟器，玩家目标是赚钱给莫矢买身体。
一阶段的主要任务是引出各位重要患者，以及让玩家玩下去，把坑挖好，在目标即将实现的时候，让玩家目标消失，玩家会发现目标无论如何也无法达到。而且是动态的无法达到。
#### 为什么会这样？ 
只要让玩家产生了这样的疑惑，一阶段的目标就完成了
### 二阶段游戏是高压锅
AI对世界罢工（虽然有点突兀吧）导致世界巨大变革，使得社会心理问题突然爆发，采用各种手段压迫玩家的视觉，增加日常工作量，增加患者痛苦，部分患者走向结局，然后主角**无可避免的走向崩溃**

### 三阶段玩家将扮演心理医生对主角进行治疗
这里采用meta要素，让玩家进行伪文件处理，进行与心理医生这一职业的**深度融合**，同时发现为何一阶段目标无法达成，见证主角的过往等各种。将主角与莫矢合二为一，从伪百合→人格合并→人格分裂，现在在合并这一里。

### 四阶段陈默将部分取代莫矢对玩家进行抉择引导

见证大部分其余患者的抉择与终局。

#### 然后陈默选择释放莫矢，人格重新分裂

### 五阶段是后日谈，AI首脑（我知道这很土，这样写是方便理解）向陈默，或者说玩家，寻求治疗，进行交谈，探讨爱与生命，陪伴与痛苦，死亡与AI。





## 可以看到，这是一个完完全全的剧情驱动游戏
