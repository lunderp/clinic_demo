# 长话长说
该项目起始于突发奇想，那天我在**我开的mc服务器(MCreatopia)里**跟管理员闲聊。
突然发现管理里面既有技术，又有美术，我当策划，岂不是刚好是独立游戏铁三角了？
聊完之后，我尝试推动项目进行，但是就我观察大家似乎并不很感冒

#### 我知道，空头支票是没有说服力的
所以我需要自己探索一段时间，游戏项目就此搁置，但**灵感**不会停止生长。回过神来，想把脑中的场景变为现实的冲动愈发强烈，无论如何，我生来就是要做游戏的

### LLM高速发展的现在，通过AI辅助决策是一件十分有效率的事情
游戏需要**剧本**、**美术**、**音乐**、**程序**，这是外显的最重要的四个要素，
而AI都能帮助这些要素的实现，我因此而对AI进行研究，期间探索了**StableDifusion(SD)、SD XL、
Pony、SD webUI、SD ComfyUI、Nano Banana、gpt-imagine2、suno**等、始终无法得到理想中的效果，我意识到，通过AI进行素材的生产无疑是跑偏了的，对于证明自己工作能力的项目，够用的素材即可。

但探索并非一无所获，就我所看到的

**通过AI辅助决策是一件十分有效率的事情**

该文档是在chatgpt的建议下进行撰写，工作留痕，git版本管理都是听从了AI的建议，我认为我行走在正确的路上。除此之外，通过AI进行**适应性学习**是一件非常有效率的事情，对于godot引擎的学习，我在b站看视频入门、询问AI获得针对性解决问题的建议、官方文档进行节点相关信息的查询，帮助我解决了所有我目前遇到的问题，其中包括我个人的**职业规划**、**优先级排序**等。
